# Windows Calculator Vue Demo

这是一个使用 Vue 3 和 Vite 实现的 Windows 标准计算器前端项目。界面参考 Windows 计算器标准模式，包含基础计算、历史记录、记忆功能和键盘输入。

## 运行项目

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

默认访问地址：

```text
http://localhost:5173
```

生产构建：

```bash
npm run build
```

## 测试

运行全部测试：

```bash
npm test
```

当前测试覆盖：

- 数字、小数、退格、清空、CE
- 四则运算、连续运算、重复等号
- 除以零、负数开平方等错误状态
- `%`、`1/x`、`x²`、`√x`、`+/-`
- 一元函数表达式显示，如 `negate(9)`、`√(√(9))`
- 历史记录生成
- 记忆功能模拟
- 键盘输入映射

## 功能清单

基础输入：

- `0-9` 数字输入
- 小数点输入
- 退格
- `C` 清空全部状态
- `CE` 清空当前输入
- `+/-` 正负切换

核心计算：

- 加、减、乘、除
- 等号计算
- 连续运算
- 重复等号
- 运算符替换
- 除以零错误提示

一元运算：

- `%`
- `1/x`
- `x²`
- `√x`
- `+/-`

表达式显示：

- 一元操作会显示函数表达式
- 示例：`negate(10)`、`sqr(5)`、`√(9)`、`1/(4)`
- 函数表达式参与历史记录

历史记录：

- 点击 `=` 后生成历史记录
- 历史记录包含表达式和结果
- 点击历史结果可回填到显示区
- 支持清空历史

记忆功能：

- `MS` 保存当前显示值
- `MR` 读取记忆值
- `MC` 清空记忆值
- `M+` 记忆值加当前显示值
- `M-` 记忆值减当前显示值
- 记忆面板可查看、回填、清空记忆值

键盘支持：

| 按键 | 功能 |
| --- | --- |
| `0-9` | 输入数字 |
| `+` | 加 |
| `-` | 减 |
| `*` | 乘 |
| `/` | 除 |
| `.` | 小数点 |
| `%` | 百分号 |
| `Enter` / `=` | 等号 |
| `Backspace` | 退格 |
| `Delete` | CE |
| `Esc` | C |

## 项目结构

```text
src/
  App.vue                     页面和交互状态
  calculatorEngine.js         计算器核心状态机
  calculatorEngine.test.mjs   核心计算测试
  calculatorFeature.test.mjs  功能场景测试
  main.js                     Vue 入口
  styles.css                  页面样式
```

## 当前范围

已实现 Windows 标准计算器的主要功能。不包含科学计算器、程序员计算器、日期计算、单位换算和窗口系统级拖拽缩放。
